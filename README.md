# www-101ta28-com

101ta28のポートフォリオサイトです。Astroで静的HTMLを生成し、Cloudflare Workers Assetsから配信します。

## Commands

- `bun install`: 依存関係をインストールします。
- `bun run dev`: Astroの開発サーバーを起動します。
- `bun run build`: 本番用の静的ファイルを`dist/`に生成します。
- `bun run preview`: 生成したサイトをローカルで確認します。
- `bun run lint`: Astroの型・構文チェックを実行します。
- `bun run deploy`: WranglerでCloudflareへデプロイします。
